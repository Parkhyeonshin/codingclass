repeat()

이 메서드는 문자열을 복사해서, 복사한 새로운 문자열을 반환합니다. IE는 Edge부터 지원합니다.

■ 문법

이 메서드는 문자열을 복사하여, 복사한 새로운 문자열을 반환합니다.

string.repeat( count )

■ 특징

▶ 반복문을 사용하여 반환도 가능합니다.
▶ 반복횟수인 count는 양의 정수여야 하며 무한대보다 적고, 최대 문자열 크기를 넘어서는 안됩니다.

■ 예시

이해를 돕기 위해 예시를 들어봅시다. 주석으로 결과값을 작성하였습니다. 

const str1 = "javascript";
const currentStr1 = str1.repeat(0);     //''
const currentStr2 = str1.repeat(1);     //javascript
const currentStr3 = str1.repeat(2);     //javascriptjavascript
