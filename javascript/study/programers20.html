프로그래머스Lv.1 : 내적

■ 문제 설명

길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 내적을 return 하도록 solution 함수를 완성해주세요. 이때, a와 b의 내적은 a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1] 입니다. (n은 a, b의 길이)

■ 제한 조건

✔ a, b의 길이는 1 이상 1,000 이하입니다.
✔ a, b의 모든 수는 -1,000 이상 1,000 이하입니다.

■ 입출력 예

a b return
10 true
[1,2,3,4] [-3,-1,0,2] 3
[-1,0,1] [1,0,-1] -2

■ 입출력 예

1. a와 b의 내적은 1*(-3) + 2*(-1) + 3*0 + 4*2 = 3 입니다.
2. a와 b의 내적은 (-1)*1 + 0*0 + 1*(-1) = -2 입니다.

■ 내멋대로 풀어헤치기😜

각 배열의 0번 째부터 마지막번 째까지 a배열의 값과 b배열의 값을 곱한 후 더합니다.

function solution(a, b) {
    let result = 0;
    for (let i in a) {
        result += a[i] * b[i]
    }
    return result
}

■ 다른사람 입맛도 맛보기 😋

function solution(a, b) {
    return a.reduce((acc, _, i) => acc += a[i] * b[i], 0);
}
