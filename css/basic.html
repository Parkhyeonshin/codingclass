font-size, 단위의 고찰


px

해상도에 따라 상대적으로 달라지는 기본 단위

font-size: 12px;


%

부모 요소의 글자 크기를 100% 기준으로 계산한 %단위

font-size: 150%;


em

부모 요소의 글자 크기를 100% 기준으로 계산한 100분의 1단위

font-size: 1.5em;


rem

최상위 요소의 크기를 100%기준으로 계산. html요소에서 지정한 글자 크기가 1rem이 됩니다.

font-size: 1.5rem;


vw, vh

너비, 높이값의 100분의 1단위

font-size: 10vw;
