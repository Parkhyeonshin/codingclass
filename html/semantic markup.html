시멘틱 마크업

시멘틱 마크업은 마크업시 의미에 부합하는 태그를 사용하라는 뜻
why? 이용자들의 환경이 모두 동일한게 아니라 모두가 똑같게 받아들일 수 있지 않기 때문
ex) 

< !DOCTYPE html >
< html lang="ko" >
< head >
            <meta charset = "UTF-8">
            <title>시멘틱 마크업</title>
< /head >
< body >
        반응형웹
< /body >
< /html >

--> 스크린 리더에서 '반응형웹'라고만 읽어서 제목인지 알 수 가 없음

< !DOCTYPE html >
< html lang="ko" >
< head >
        <meta charset = "UTF-8">
        <title>시멘틱 마크업</title>
< /head >
< body >
        <h1>반응형웹</h1>
< /body >
< /html >

--> 첫 제목을 의미하는 <h1>태그 덕분에 '반응형웹 헤딩 일'이라고 읽어줌
