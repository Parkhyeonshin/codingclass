HTML 기본 문서

태그 : 명령어의 형태가 '<>'되어있는 것. ex _ <html>, <head>, <body>,
요소 : 태그의 시작(<>)과 종료(</>)
마크업 : 요소들로 코딩한 것
속성 : <html lang="ko">,<meta charset="UTF-8">처럼 태그 내부에 부가적인 설정값을 선언하는 것 

<!DOCTYPE html>
<html lang="KO">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
</body>
</html>

<html> 요소는 <head> 요소와 <body> 요소로 구성되어 있으며, <head> 요소 안에는 <meta>, <title>, <link>, <style>, <script>등의 요소들이 올 수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작성자 등을 지정합니다.
<title> HTML 문서 제목 지정.
<link> 외부 CSS파일을 HTML 문서에 연결할때 사용.
<style> HTML 문서 내 스타일 (CCS)를 정의.
<script> HTML 문서 내부에서 자바스크립트를 정의하거나, 외부 자바스크립트 파일 불러올 때 사용.

lang 속성 : 문서에서 사용하는 언어, 스크린리더(음성낭독 프로그램)이 읽어줌. 꼭 정의해야 하는 코드
ex_1, 2, 3의 경우 lang="ko"로 지정하면 '일, 이, 삼'으로, lang="en"으로 지정하면 '원, 투, 쓰리'

<i lang="de">Ich liebe dich</i>

뷰포트 : 문서를 조절해 주는 속성이다.

<meta charset = "URF-8">
<meta name = "viewport"
content = "widt=device-wiedth, initial - scale = 1.0, minimum-scale=1.0,maximum-scale=1.0,user-
scalable=no">

<meta>태그의 charset 속성의 값 UTF-8을 문자 인코딩(charset) 방식이라고 하는데, UTF-8을 주로 사용
