.blind{
     position: absolute;
    clip: rect(0 0 0 0);
    width: 1px;
    height: 1px;
    margin: -1px;
    overflow: hidden;
}

대제목에 블라인드 효과를 주어 감추었습니다.큰 주제이지만 구조상 밑으로 내려가니깐 웹표준을 준수하기 위해서 이렇게 한게 아닐까...?싶습니다.

.imgText__txt ul li::before{
    content: '';
    width: 5px;
    height: 5px;
    border-radius: 50%;
    position: absolute;
    left: 5px;
    top: 9px;
    background: #666;
}

li태그에서 기본 제공하는 아이콘을 사용하지 않고 ::before가상요소를 통해서 좀 더 이쁜 단락기호로 표시해주었습니다. 그리고 이런 속성으로 부여하면 글자와 겹치게 되는데 li에 'padding-left'를 주어 배치를 해주었습니다.
